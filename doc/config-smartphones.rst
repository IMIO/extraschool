Configuration des smartphones
=============================

.. toctree::
   :maxdepth: 3
   
#############
Introduction.
#############

Etapes à suivre pour la configuration des smartphones à l'application extraschool


#########
Prérequis
#########

* Créer un compte Google pour la commune (Demander la structrure car c'est toujours la même qui est réalisée)
* Communiquer le mot de passe au responsable de l'application chez Imio. (Ou demander à Imio la création du compte Google, ce qui est plus simple)
* Il faut que l’utilisateur du smartphone, dans l'application extrascolaire, ait les droits d’accès pour que la synchronisation se déroule correctement. (AES utilisateur)


#############
Configuration
#############

* Ouvrir le téléphone avec le code SIM (Attention faire attention à garder le bon téléphone avec la bonne carte SIM)
* Mettre le code du Wi-Fi (si la connexion ne s’effectue pas : partager la connexion de son téléphone.)
* On colle des étiquettes avec le nom des lieux d’accueil sur les boites des téléphones et sur l'arrière du smartphone.
* On Installe Barcode Scanner disponible dans Google play
* On scanne le premier QR Code (identique dans toutes les fiches des lieux d’accueil) afin d’installer l’application sur les smartphones.
* Il est intéressant de vérifier que le téléphone soit à la bonne heure.
* Il faut se rendre dans l’application extrascolaire préalablement installée:
   * Menu
   * Configuration (Mot de passe : 432432)
   * QR Config : Scanner le deuxième QR Code pour prendre les informations de configuration de l’application du lieu d’accueil.
   * Enregistrer
   * Quitter car après l'enregistrement il ne le fait pas automatiquement
* Menu → Transmettre les données.
* Installer Teamviewer Quick Support (Ouvrir l'application) et s’il  demande d’installer une extension l’installer aussi.
* Paramètres du téléphone → Mise à jour menu → Paramètres de mise à jour → Annuler  (on enlève les mises à jours car cela n’est pas utile pour le téléphone)
* Enlever le code SIM : Dans les paramètres → Sécurité et désactiver la carte SIM (le re-noter et ne pas en mettre)  : on garde quand même les codes dans la boites et souvent on le note dans le nom dans la définition des smartphones dans l'application extrascolaire
* Paramètres → Gestion des applications → Paramètres du lanceur → il faut définir “Garderie”
* Dans les Paramètres (on va dans le menu en haut : glisser avec le doigt où il y a le Wi-Fi,... ) → Garderie : Toujours


##################
Dans l'application
##################

Dans les fiches que l’on créé dans l'application AES pour les smartphones :

* On peut les dupliquer afin de ne pas toutes les encoder
* Il faut faire attention de changer le temps de transmission (de décaler le temps de 5 minutes)
* Bien faire correspondre le smartphone avec le lieu d'accueil.
