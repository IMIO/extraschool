Un remboursement
================

.. toctree::
   :maxdepth: 3

Il arrive que le parent effectue des prépaiements afin de ne plus se soucier des factures de l'activité extra-scolaire. Lorsque le parent
ne possède plus d'enfants dans les différentes écoles de la commune et qui lui reste du prépaiement, il faut le rembourser.


Pour effectuer cette manipulation, il faut se rendre sur la fiche signalétique du parent.

Dans l'onglet "Prépaiement", il y a un bouton nommé "Remboursement".
Si on clique sur ce bouton, cela va :

* Mettre le total de prépaiement de ce parent à zéro.
* Générer dans les facturiers un document pour le parent afin d'avoir une preuve de ce remboursement.

Cette méthode n'est pas encore optimale. Nous allons donc l'améliorer au besoin.
N'hésitez pas à nous faire votre retour sur ce processus afin d'ajouter des éléments qui seraient intéressants pour toutes les communes.
